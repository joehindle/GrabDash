# GrabDash

GrabDash is a basic Flask to-do app with SQLite and SQLAlchemy. It lets you add, update, and delete tasks from a simple web interface.

## Features

- Add tasks
- View tasks
- Update tasks
- Delete tasks

## Tech Stack

- Python
- Flask
- Flask-SQLAlchemy
- SQLite

## Run Locally

1. Install dependencies:

```bash
pip install -r requirements.txt
```

2. Start the app:

```bash
python index.py
```

3. Open the app in your browser:

```text
http://127.0.0.1:8000
```
